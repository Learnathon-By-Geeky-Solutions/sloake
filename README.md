## Getting Started

### Prerequisites

- Node.js >= 14.0.0
- MongoDB >= 4.4
- PostgreSQL >= 13

### Installation

1. Clone the repository

```bash
git clone git@github.com:Learnathon-By-Geeky-Solutions/sloake.git
cd sloake
```

2. Install dependencies

```bash
npm install
```

3. Start development

```bash
npm run dev
```
